## English

### 📱 JingHu - AI-Powered Spam Call &amp; SMS Blocker

Protect your privacy with on-device AI. No data uploads, 100% local processing.

#### ✨ Features

- **🔒 Privacy-First**: All processing happens on your device. Nothing is uploaded.
- **🤖 AI-Powered**: MobileBERT model for intelligent spam detection
- **📞 Call Blocking**: Automatically screen and block spam calls
- **💬 SMS Filtering**: Detect and filter junk messages
- **🔄 Cloud Sync**: Backup your blocklists securely (Premium)
- **⚡ Fast Performance**: &lt;500ms response time

#### 📥 Download

| Version | Size | Download |
|---|---|---|
| **Google Play (Firebase)** | 24.1 MB | [Download](https://github.com/782A/jinghu/releases/latest) |
| **F-Droid (Pure Open Source)** | 15.98 MB | [Download](https://github.com/782A/jinghu/releases/latest) |

#### 🚀 Getting Started

1. Download and install JingHu
2. Grant necessary permissions (Phone, SMS)
3. Enable call screening in your phone settings
4. You're protected!

#### 🛠️ Tech Stack

- **Language**: Kotlin
- **UI**: Jetpack Compose
- **ML**: TensorFlow Lite (MobileBERT)
- **Database**: Room + SQLCipher
- **Architecture**: MVVM + Clean Architecture

#### 📈 Roadmap

- [x] Core call &amp; SMS blocking

- [ ] Multi-language support
